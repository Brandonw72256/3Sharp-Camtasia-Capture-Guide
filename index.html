Getting started with Camtasia
Follow these steps to download and install Camtasia before continuing.
1
Download Camtasia
Go to techsmith.com/download/camtasia and download. Sign in with your licensed account. If you don't have a license, a free 30-day trial is available .
2
Install and open Camtasia
Run the installer and open Camtasia once it's complete. Sign in with the account you just created.
What type of demo are you recording?
This determines which environment settings you'll need to configure.
Single app
Choose this if your demo stays within one application or browser tab the whole time.
e.g. Walking through Microsoft Teams.
Multiple apps
Choose this if your demo stays in a browser but switches between several tabs.
e.g. A demo showcasing multiple browser apps such as Teams, Excel, and Word.
Desktop apps
Choose this if your demo moves between two or more native Windows applications.
e.g. Switching between Teams, Outlook, and Edge.
Prepare your environment
Set up your screen before recording. Check each item off as you go — all are optional but recommended.
Start your Camtasia recording
Follow these steps in Camtasia to capture your demo.
1
Open Camtasia and start a new recording
In Camtasia, select New Recording. Toggle screen recording to ON.
2
Select the correct display
In the recording toolbar, ensure the screen set to the 16:9 ratio is selected. If not, use the dropdown to select the correct display. Do not manually adjust the resolution numbers in Camtasia — change resolution in Windows Display Settings instead.
3
Optional: enable microphone
If you'd like to narrate as you record, toggle the Microphone ON. This helps with talk track placement later, but is not required.
4
Turn off Camtasia Rev
Ensure Camtasia Rev is toggled OFF before recording.
5
Hit record
Select Record and step through your demo click by click. Pause a few seconds between each step if possible.
6
Export and send
When finished, export via File → Export → Zipped project. This saves the full project as a single zip file. Send it to your 3Sharp contact.
You're all set!
Your environment is configured and your recording is underway. Send the zipped Camtasia project to your 3Sharp contact when you're done.
